O que é o Instagram?

O Instagram é uma rede social e um serviço online gratuito focado no compartilhamento de fotos e vídeos. Lançado em 2010 pelos brasileiros Mike Krieger e Kevin Systrom, ele permite que os usuários publiquem fotos e vídeos, aplicando filtros, adicionando legendas e hashtags para torná-los pesquisáveis.

O que é o GitHub?

O GitHub é uma plataforma online que utiliza o sistema de controle de versão Git para hospedar, gerenciar e colaborar em projetos de software, funcionando como um hub para que desenvolvedores de todo o mundo compartilhem código e trabalhem juntos.

O que é o LinkedIn?

O LinkedIn serve para construir e expandir uma rede de contatos profissionais, encontrar oportunidades de emprego, divulgar o histórico profissional como um currículo online, aprender e compartilhar conhecimentos, fortalecer a marca pessoal e empresarial, e promover a interação e troca de experiências no mercado de trabalho.
